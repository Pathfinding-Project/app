# Waypoint

Waypoint is a visualiser and debugging tool for pathfinding
search.

[Open Waypoint](https://path-visualiser.github.io/app)

## Start Development Server

```bash
git clone https://github.com/path-visualiser/app
cd ./app
npm i
npm start
```
